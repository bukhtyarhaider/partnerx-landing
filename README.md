# PartnerX Landing Page

This is a free and open-source landing page project created by **Bukhtyar Haider**.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4
- Lucide React (Icons)
- React Router DOM

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Project Structure

- `src/components`: Reusable UI components (Navbar, Footer, Hero, Features, etc.)
- `src/pages`: Page components (Home, About, Pricing, FeaturesPage, Legal)
- `src/index.css`: Global styles and Tailwind configuration

## Styling & Design

- Fully migrated to **Tailwind CSS v4** with utility‑first approach.
- Dark theme with custom color palette (`--color-primary`, `--color-primary-hover`, `--color-accent`, etc.).
- Glassmorphism effects applied via the `.glass` utility class for cards and sections.
- Responsive layout and smooth micro‑animations for an engaging user experience.
